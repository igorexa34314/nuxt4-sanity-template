# Nuxt 4 + Sanity template

A simple monorepo consisting of an Nuxt 4 app and Sanity.io studio.

---

### Build Setup

```bash
# install app & studio
$ yarn install

# serve app at localhost:3000
# serve studio at localhost:3333
$ yarn dev

# generate static project and launch app
$ cd ./packages/app
$ yarn generate
$ yarn serve

# deploy studio
$ cd ./packages/studio
$ yarn deploy
```
